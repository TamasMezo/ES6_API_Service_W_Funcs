# Express & NodeJS starter with basic functionalities
## Small API project ES6
Features:
  - authentication: 
    - sign up
    - login
    - password reset
  - creating collections
  - creating instances
  - deleting instances
  - requesting information from DB
  - updating DB data
  - cronjob
  - logging events
  - sending emails (nodemailer)
  - streams 
  - excel export (sheet.js)
  - google sheets API
  - google drive API
